# 华夏：文明崛起

一款以华夏五千年历史为背景的文明模拟游戏，灵感来源于《矮人要塞》等经典建造模拟游戏。

## 项目简介

《华夏：文明崛起》是一款基于 HTML5 Canvas 的单文件文明模拟游戏。玩家将扮演华夏部落的首领，带领族人从蛮荒时代一步步发展壮大，经历青铜时代、铁器时代，最终建立不朽的文明。

## 游戏特性

### 🏛️ 核心玩法
- **文明建设**：建造房屋、农田、作坊、学堂等20+种建筑
- **族人管理**：士农工商四大职业体系，每位族人都有独立的属性和成长
- **资源经济**：粮食、木材、石料、青铜、学识等多种资源的生产与调配
- **时代演进**：从蛮荒时代到青铜时代，解锁更多科技与建筑

### ⚔️ 军事系统
- **军队编制**：战士、弓箭手等兵种
- **出征讨伐**：主动出击消灭野兽和敌对势力
- **防守巡逻**：保护领地安全
- **势力外交**：与周边部落通商、结盟或宣战

### 📚 科技树系统
- **四大分支**：基础技艺、工具建造、军事战争、文化制度
- **17项科技**：从钻木取火到文字发明
- **时代跨越**：研究关键科技进入新时代

### 🌤️ 天气与灾害
- **6种天气**：晴天、阴天、雨天、暴雨、酷暑、雪天
- **天气影响**：不同天气对生产、移动、健康的影响
- **5种自然灾害**：洪水、干旱、瘟疫、地震、蝗灾
- **动态环境**：灾害与天气联动，考验玩家应对能力

### 🏰 领地系统
- **5级领地**：村落→集镇→城邑→都城→王城
- **建设限制**：建筑只能在领地范围内建造
- **影响力扩张**：通过积累影响力升级领地
- **可视化边界**：蓝色虚线标示领地范围

### 🎮 操作体验
- **三国志风格鼠标提示**：悬停显示坐标、地形、人物、敌人详细信息
- **边缘滚动**：鼠标移到屏幕边缘自动移动地图
- **键盘快捷键**：1-6快速切换工具，WASD移动地图
- **变速控制**：暂停/正常/快速/极速四档游戏速度

## 快速开始

### 运行方式
直接用浏览器打开 index.html 即可开始游戏，无需任何依赖安装。

```bash
# 方式一：直接双击打开
双击 index.html

# 方式二：本地服务器（推荐）
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 新手指引
1. **安家落户**：先建造几座房屋让族人有栖身之所
2. **发展农业**：开垦农田保证粮食供应
3. **科技研究**：建造学堂研究科技提升效率
4. **建立武备**：建造兵营训练战士保卫家园
5. **扩张领地**：积累影响力升级领地扩大建设范围

## 技术实现

- **纯前端实现**：单 HTML 文件，HTML5 Canvas 渲染
- **无外部依赖**：不依赖任何第三方库
- **程序化生成**：噪声算法生成地形地图
- **实体AI系统**：基于状态机的族人行为系统
- **实时模拟**：需求驱动的经济和生态模拟
