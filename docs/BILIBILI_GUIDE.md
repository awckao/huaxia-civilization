## 三、录制技巧提示

### 3.1 录制软件推荐

| 软件 | 优点 | 适合场景 |
|------|------|----------|
| **OBS Studio** | 免费开源，功能强大，支持场景切换 | 推荐首选，可添加摄像头、字幕等 |
| **班迪录屏 (Bandicam)** | 体积小，帧率高，游戏录制友好 | 追求高画质流畅度 |
| **NVIDIA ShadowPlay** | 硬件加速，几乎不占性能 | N 卡用户首选 |
| **Xbox Game Bar** | Win10/11 自带，无需安装 | 临时快速录制 |

> 录制参数建议：分辨率 1920×1080，帧率 30fps 或 60fps，码率 8-15Mbps，格式 MP4（H.264）。

### 3.2 游戏内设置建议

1. **浏览器设置**：
   - 使用 Chrome 或 Edge 浏览器
   - 全屏模式（按 F11）录制，避免工具栏干扰
   - 缩放调整到 100%，保证画面清晰

2. **游戏内调整**：
   - 录制前先玩 10-15 分钟，发展到一定阶段再开始录（避免开局空白期太长）
   - 准备两个存档：一个蛮荒时代开局（展示新手期），一个青铜时代（展示进阶内容）
   - 游戏速度可随时调整（空格暂停，加速展示发展过程）
   - 适当放大/缩小地图，保证观众看清楚细节

3. **画面优化**：
   - 录制前刷新页面，确保游戏运行流畅
   - 关闭浏览器其他标签页，减少性能占用
   - 使用深色主题的浏览器，避免画面不协调

### 3.3 剪辑要点

1. **节奏把控**：
   - 前 30 秒必须有爆点（时代跨越动画、战斗画面等）
   - 建造过程可以 2-4 倍速快进，节省时间
   - 重要操作（如研究科技、时代跨越）用原速，配合解说
   - 每 1-2 分钟一个小高潮，保持观众注意力

2. **解说录制**：
   - 建议先录画面，后期配音（更容易控制节奏和口误）
   - 录音设备：至少要有个基础麦克风，避免笔记本内置麦的杂音
   - 录音环境：安静房间，避免回声和背景噪音
   - 音频格式：WAV 或 320kbps MP3，采样率 44100Hz

3. **剪辑软件推荐**：
   - 专业级：Adobe Premiere Pro、DaVinci Resolve（免费）
   - 入门级：剪映专业版（免费，功能足够，B站友好）
   - 特效：AE（可选）或剪映内置特效

4. **BGM 选择建议**：
   - 主 BGM：中国风纯音乐（古筝、竹笛、二胡等）
   - 战斗段落：节奏快、鼓点重的古风战斗音乐
   - 灾害段落：紧张压抑的氛围音乐
   - 注意版权：使用 B 站音乐库或无版权音乐，避免侵权

### 3.4 封面制作建议

1. **工具**：
   - 专业：Photoshop
   - 简单：Canva（可画）、稿定设计
   - 免费：GIMP、Paint.NET

2. **素材来源**：
   - 游戏实机截图（自己玩的时候截几张好的）
   - 官方宣传图（docs 目录下的 promo-banner.jpg 等）
   - 字体：站酷快乐体、汉仪尚巍手书等有书法感的字体

3. **构图法则**：
   - 左图右字或上图下字，信息层次清晰
   - 主标题字号要大，远看也能看清
   - 颜色不超过 3 种主色调，避免杂乱
   - 人物/建筑主体不要被 B 站播放按钮遮挡（注意安全区域）
