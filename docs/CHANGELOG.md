# 更新日志

所有重要的版本更新都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [0.4.0] - 2026-07-11

### 新增
- 🏛️ 领地系统：5级领地等级（村落→集镇→城邑→都城→王城）
- 🏰 建设限制：建筑只能在领地范围内建造
- 📈 领地升级：消耗影响力扩张领地范围
- 🖱️ 三国志风格鼠标提示：坐标、地形、领地、人物、敌人信息
- 🌤️ 天气系统：6种天气（晴、阴、雨、暴雨、酷暑、雪）
- 🌊 自然灾害系统：洪水、干旱、瘟疫、地震、蝗灾
- ⛈️ 天气灾害联动：暴雨引发洪水，酷暑引发干旱
- 🎯 边缘滚动：鼠标移到屏幕边缘自动移动地图
- ⚔️ 初始战士：游戏开始时已有3名战士
- 💥 战斗特效：浮动伤害数字

### 优化
- 出征算法优化：优先派遣最近、最健康的战士
- 战斗反馈增强：出征特效、失败提示
- 势力袭击：敌对势力每月有几率发动袭击
